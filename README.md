# ai-brand-monitor

Monitor your brand's visibility across AI search surfaces.

Tracks how your brand appears in Google AI Mode, AIO, ChatGPT, Gemini, Perplexity, and Claude.

## Quick Start

1. Clone this repo into your `.claude/skills/` directory
2. Copy `examples/config.example.yaml` to `config.yaml`
3. Edit `config.yaml` with your brand details
4. Run `/ai-brand-monitor` in Claude Code
